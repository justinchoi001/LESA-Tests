LESA - Customer Portal - Liferay Portal

Prerequisites:

  • Liferay CSE account

Description:

This test is designed to have users verify the basic functionality of customer portal. Users will verify specific links for downloading Liferay Portal are valid. Users will download fix packs, patching tool, and security updates.

Road Map:

  1. Sign In As Liferay CSE
  2. Assert Documentation
  3. Assert Patching Tool

Test Steps:

  1. Navigate to http://www-uat.liferay.com
  2. Click the Main Menu drop down
  3. Click Customer Portal
  4. Assert that the Liferay Portal icon displays
  5. Assert that the Social Office icon displays
  6. Assert that the Liferay Sync icon displays
  7. Assert that the Liferay Faces icon displays
  8. Assert that the Developer Studio icon displays
  9. Assert that the Mobile SDK icon displays
  10. Assert that the Liferay Portal 6.2 box displays
  11. Assert that the Liferay Portal 6.1 box displays
  12. Assert that the Liferay Portal 6.0 box displays
  13. Assert that the Liferay Portal 5.2 box displays
  14. Assert that the Top Articles section displays in the bottom left
  15. Assert that the Top Downloads section displays in the bottom middle
  16. Assert that the Latest News section displays in the bottom right
  17. Click Liferay Portal
  18. Assert that the latest version of Liferay Portal displays at the top
  19. Assert that the downloads drop down display
  20. Assert that the Download EE (latest version) displays
  21. Assert that the Archive link displays
  22. Assert that the Additional Files link displays
  23. Assert that the Support Matrix box displays
  24. Assert that the Support Information box displays
  25. Assert that the Deployment Checklist box displays
  26. Assert that the License Key box displays
  27. Assert that the Knowledge Base Box displays
  28. Assert that the Documentation box displays
  29. Assert that the Patching Tool box displays
  30. Assert that the Security Patches box displays
  31. Assert that the Patches box displays
  32. Click Find Answer in the Knowledge Base box
  33. Assert the "Welcome to the Knowledge Base" message displays
  34. Assert the Search field displays
  35. Click Back in the browser
    Assert Documentation
  36. Click Learn More in the Documentation box
  37. Assert that the User Guide is available for download
  38. Assert that the Developer Guide is available for download
  39. Assert that Release Notes are available for download
  40. Click Back in the browser
    Assert Patching Tool
  41. Click Download in the Patching Tool box
  42. Assert that the latest patching tool is downloaded
  43. Click Learn More in the Security Patches box
  44. Assert that the latest LSV fixes are available for download
  45. Click Learn More for the latest LSV
  46. Click the Download button
  47. Follow the additional prompts since LSVs are included in Platform fix packs
  48. Click the Main Menu drop down
  49. Click Customer Portal
  50. Click Liferay Portal
  51. Click Download all Patches in the Patches box
  52. Assert that the latest fix packs are downloaded
  53. Click the Administration 4 Release Notes link
  54. Assert the Release Notes title displays
  55. Assert that Change Log displays
